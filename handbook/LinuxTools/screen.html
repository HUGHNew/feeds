# screen

GNU 协议 apt install screen 配置文件 .screenrc

screen 的指令有两种

  • screen内外都能用的命令行参数(screen [options])
  • screen内部的组合键控制(默认以 Ctrl+A 开始的组合键)

# 使用

# 创建

screen
screen -S name
# 创建新的screen -S 指定会话名 否则默认生成一个

# screen command
screen ssh user@ip
# 创建新的screen并执行命令
Ctrl + A, C
# 可以创建新的会话
